Simple Animation

This project covers the fundamentals of Communcation & Presentation, Interactive Development and Design and Visualisation.


This project was created through the use of Javascript and CSS. The animations were created in the stylesheet with keyframes which would be run when the button elements were clicked. The script has querySelectors to apply clickListener functions to the buttons. Once the buttons are clicked and the clickListener functions run, the CSS animation classes are appended to the existing classes, thus animating the character.
